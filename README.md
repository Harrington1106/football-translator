# ⚽ 足球新手村 — Football Translator

> 世界上最适合足球新手的世界杯网站。3分钟看懂一场比赛。

**https://football-translator.vercel.app**

## 这是什么

不做传统足球数据网站。不堆比分、阵容、xG。而是做一个 **"足球翻译器"**——帮助从未看过足球的人在 3 分钟内理解一场比赛。

## 功能

| 功能 | 说明 |
|---|---|
| 🏠 首页 | 正在直播、今日最值得看、即将开始、淘汰赛对阵图、射手榜 |
| 📊 比赛详情 | 实时比分、事件时间线、实力对比、新手观赛指南 |
| 🤖 AI 解说 | DeepSeek V4 Pro 实时回答"刚刚发生了什么""为什么大家在欢呼" |
| 🌍 球队页 | 48 队完整阵容、明星球员、比赛记录、FIFA 排名 |
| ⭐ 球员页 | 球员介绍、代表时刻、新手认识原因 |
| 📚 足球课堂 | 越位、点球、红黄牌、补时等 8 课 |
| 🌓 主题 | 日间/夜间/跟随系统 |

## 技术栈

- **Next.js 16** + TypeScript
- **TailwindCSS 4** + shadcn/ui
- **Framer Motion**
- **worldcup26.ir** — 免费无 key 的 2026 世界杯实时 API（104 场比赛、比分、进球者）
- **DeepSeek V4 Pro** — AI 解说和阵容生成

## 本地运行

```bash
npm install
# 添加 .env.local:
# DEEPSEEK_API_KEY=sk-your-key
npm run dev
```

## 部署

一键部署到 Vercel，需设置环境变量 `DEEPSEEK_API_KEY`。

## 设计理念

- **不展示数据，先解释意义** — "控球率 63%"→"法国大部分时间掌控比赛"
- **不学术语** — "Expected Goals"→"预计进球概率"
- **每个页面回答用户真正的问题** — "为什么重要？""我应该关注谁？""刚刚发生了什么？"
